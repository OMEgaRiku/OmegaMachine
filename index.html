■ 基本・比較
[属性]と[数字]の関係 ( <, =, > )
例: 「炎と3の関係」など。
そのルーンが指定された数字より大きいか小さいか 同一か。
判定: 炎が4でTRUEなら「炎 > 3」
[属性]と[属性]の比較 ( <, =, > )
例: 「炎と水の比較」など。
2つのルーンの大きさを比べる。
判定: 炎4・水2でTRUEなら「炎 > 水」
順序の法則 (昇順, 降順, 順不同)
3つの数字の並び方。
昇順: 1-2-5 (増えていく)
降順: 5-3-2 (減っていく)
順不同: 1-5-5 (バラバラ)
最大値の所在 (≧) / 最小値の所在 (≦)
誰が一番大きい(または小さい)か。
判定: 5-2-2でTRUEなら「炎が最大」
■ 数学・合計
[属性]の偶奇 (偶数, 奇数)
その数字が2で割り切れるかどうか。
判定: 炎が4でTRUEなら「炎=偶数」
合計の偶奇 (偶数, 奇数)
3つの合計値が偶数か奇数か。
判定: 1+2+4=7 でTRUEなら「合計は奇数」
合計と[数字]の関係 ( <, ≧ )
例: 「合計と10の関係」など。
3つの合計が基準(6や10)より大きいか小さいか。
判定: 合計11でTRUEなら「合計 ≧ 10」
合計が特定の倍数(2, 3, 5)
合計値がその数で割り切れるか。
判定: 合計10でTRUEなら「5の倍数」「2の倍数」
[属性]+[属性]と5の関係 ( <, =, > )
例: 「炎+水と5の関係」など。
指定された2つの和が5より大きいか小さいか。
判定: 3+3=6でTRUEなら「炎+水 > 5」
合計値が素数かどうか
合計が 3, 5, 7, 11, 13 のいずれかである。
判定: 合計5でTRUEなら「合計は素数」
素数の有無 (あり, なし)
数字の中に素数(2, 3, 5)が1つでも含まれるか。
判定: 1-1-4 でTRUEなら「素数なし 」
■ 構造・パターン
[数字]の数 (0, 1, 2, 3)
例: 「偶数の数」「1の数」など。
条件に合う数字がいくつあるか。
判定: 2-3-5 でTRUEなら「偶数が1個」
隣接する連番の数(0 , 1 , 2)
隣り合う数字の差が「1」である箇所の数。
4-5-1 (4と5) → 1箇所
1-2-3 (1と2, 2と3) → 2箇所
1-3-5 (繋がりなし) → 0箇所
2つの同じ数字 (なし, 1組)
ペアが含まれているか (ワンペア)。
判定: 2-2-5 (1組) / 1-2-3 (なし)
※1組の場合2-2-2を入れてもTRUEを返します。
最大と最小の差(0 ,1 ,2 ,3 ,4)
一番大きい数 - 一番小さい数。
判定: 2-5-5でTRUEなら 5-2=3 なので「差が3」
